## Image Specifications:

- Format: JPG or WebP
- Recommended size: 1200x800px or similar aspect ratio
- File size: Optimized for web (under 500KB recommended)
- Content: Should represent the city (skyline, landmark, cityscape, etc.)

Please add the appropriate images to this directory for each city.
